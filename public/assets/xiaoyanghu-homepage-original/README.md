# Xiaoyanghu Homepage Original Assets

本目录保存了 `https://xiaoyanghu.com/` 首页中公开加载的原始素材，覆盖用户提供的 7 张截图。

## 下载结果

- 图片：106 个，已下载可访问的最高原始分辨率
- 视频：2 个
- 音频：2 个
- 字体：6 个
- 页面内嵌 SVG：25 个
- 总资源：141 个

## 目录结构

- `images/`: 原始图片、贴纸、项目截图、设备模型、照片和纹理
- `videos/`: 首页循环视频
- `audio/`: 页面音频
- `fonts/`: 页面加载的 WOFF2 字体
- `svg/`: 从页面提取并展开引用的内嵌图标
- `previews/`: 按页面区域生成的联系表
- `manifest.json`: 每个文件的本地路径、原始网址和文件大小
- `catalog-by-section.json`: 按页面区域整理的图片目录
- `catalog-by-section.csv`: 可在表格软件中搜索的图片目录
- `metadata/source-inventory.json`: 浏览器提取的原始资源记录
- `metadata/dom-media-map.json`: 页面位置、上下文和图片尺寸映射
- `SHA256SUMS`: 141 个原始素材文件的完整性校验值

## 查找素材

图片文件沿用原站的哈希名称。推荐先打开 `previews/` 目录确定所在区域，再在 `catalog-by-section.csv` 中按区域、尺寸或上下文查找对应本地文件。
