# Xiaoyang Hu Inspired Style Kit

这是一套为当前作品集整理的风格相近素材，不包含 `xiaoyanghu.com` 的原始图片、视频或音频副本。

参考站点审计范围：

- 首页：复古电脑主视觉、项目设备模型、实验项目拼贴、工具包拼贴、经历徽章、个人摄影、拍立得互动、视频与音频
- `/projects`：项目缩略图和 GIF
- `/akool`：案例研究截图、流程图、响应式界面、短视频
- `/siemens`：产品流程图、界面演示和短视频

## 新增素材

### Photos

- `photos/retro-computer-hp250.jpg`: 复古设备、工具包或经历拼贴
- `photos/instant-camera-supercolor.jpg`: 拍立得互动或爱好区
- `photos/theatre-stage.jpg`: 剧场爱好与个人故事
- `photos/forest-motion.jpg`: 抽象自然、户外与过渡背景
- `photos/seattle-pike-street-bw.jpg`: 街头摄影与黑白拼贴
- `photos/architecture-model.jpg`: 建筑背景、系统思维或教育经历

### Ready-to-use Polaroids

- `frames/polaroid-architecture.webp`
- `frames/polaroid-forest.webp`
- `frames/polaroid-seattle.webp`
- `frames/polaroid-theatre.webp`

空白透明框：

- `frames/polaroid-landscape.png`
- `frames/polaroid-portrait.png`

### Device Frames

- `frames/laptop-frame.png`: 屏幕开口为 1280 x 725
- `frames/phone-frame.png`: 屏幕开口为 468 x 1043
- `frames/tablet-frame.png`: 屏幕开口为 960 x 734

这些是透明 PNG，可以把现有项目截图放在底层，再把设备框放在上层。

### Decorative Objects

- `frames/silver-medallion-gw.png`: 本项目定制的银色姓名徽章
- `frames/tape-blue.png`, `frames/tape-yellow.png`: 拼贴胶带
- `stickers/*.png`: 8 个 256 x 256 的 Fluent Emoji 3D 贴纸

### Motion and Audio

- `motion/retro-greeting-loop.webp`: 由本地 `gaoyi-wu-hero-greeting.webp` 制作的轻微 CRT 闪烁循环
- `audio/camera-shutter-holga.ogg`: 拍立得快门音效
